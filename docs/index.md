# Terefo Heal Reboa

*Terefo Heal Reboa* is a living research book about practical agent engineering: how Hermes, OpenClaw, context systems, memory, verification, and scheduled loops turn model calls into governed work.

This public site contains the reader-facing manuscript. It deliberately separates book prose from the internal apparatus that produces it. Source registries, claim ledgers, operational runbooks, editor instructions, cron reports, and chapter briefs remain part of the project workflow, but they are not presented here as chapters.

The book is still developing. Stronger chapters make cautious claims from supported sources; early research lanes are labelled as such until they have enough evidence, examples, and argument to become full chapters.

Start with the [Preface](book/preface.md), [Introduction](book/introduction.md), or [Methodology](book/methodology.md).
