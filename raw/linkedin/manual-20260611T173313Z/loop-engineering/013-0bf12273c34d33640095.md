There’s a shift happening in how engineers work with AI, and it’s being called loop engineering. Boris Cherny, who leads Claude Code at Anthropic, put it simply: he doesn’t prompt Claude directly anymore, he writes loops that prompt Claude, run tools, check results, and keep iterating until the task is done. It’s a real mindset change, from treating an LLM as a one-shot assistant to treating it as a component inside an autonomous system with its own feedback loop. As someone working in agentic AI workflows, this resonates a lot, the real value is in designing the loop, not just the prompt. Curious how others are thinking about this shift in their own work.
