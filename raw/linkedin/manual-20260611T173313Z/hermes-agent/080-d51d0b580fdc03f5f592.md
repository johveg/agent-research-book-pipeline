Nous Research lanzó el Profile Builder para Hermes Agent: una interfaz web que unifica la creación de agentes de IA especializados en un solo flujo. El builder guía al usuario para definir la identidad del agente, seleccionar el modelo (de proveedores como OpenAI, Anthropic o NVIDIA), activar habilidades integradas, instalar otras desde un hub y conectar servidores MCP (un protocolo para integrar herramientas externas). Todo corre localmente, sin que los datos salgan de la máquina. Un perfil creado es un agente aislado, con su propia memoria y configuración, listo para ejecutarse. Lectura ejecutiva: · La complejidad operativa, no la capacidad técnica, es la mayor barrera para adoptar IA agentiva en la empresa. Herramientas como esta convierten un proceso de desarrollo en uno de configuración, reduciendo drásticamente el tiempo y expertise requeridos. · El concepto de 'perfil aislado' es crítico para casos de uso reales. Permite tener un agente para análisis de datos, otro para atención al cliente y otro para monitoreo de sistemas, sin que se mezclen sus contextos o accesos, similar a tener empleados especializados en departamentos separados.
