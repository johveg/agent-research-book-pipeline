"Loop engineering" is a term that has emerged recently. It means designing the system that runs a coding agent, instead of prompting the agent yourself. The loop discovers the work, attempts it, checks the result against a testable definition of done, corrects itself on the feedback, and has a separate agent verify the output before it stops. Your job moves from writing the code to writing the system that produces it. Boris Cherny, creator and head of Claude Code at Anthropic, sums it up in one line: "My job is to write loops."
